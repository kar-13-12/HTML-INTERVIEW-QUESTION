2) Explain br tag?

  1. br : Line Break
  2. Slef Closing Tag
  3. It is a Block Element
  4. All Element Occurring After The "br" Will be sent to next line.
  5. Used for creating a division of Line.
  6. When the web page is rendered in a web browser, any content that comes after the br"tag will appear on thenext line.

3) Explain how many different type of lists we have in html?

  1. Ordered List : "ol"
    • Type : 1/I/i/A/a
    • Default : 1
    • Reversed
    • Start : ""

  2. Unordered List : "ul"
    • Type : Disc / Square / Circle / None
    • Default : Disc

  3. Description List : "dl"
    • dt - Description Title
    • dd - Description Data

4) What are Semantic tag?

HTML Tags can be classific into two group

  1. Semantic Tag
    • Tags that are specific about their functionality.
    • Eg : img tag, button tag.
  2. Non-Sementic tag
    • Tags that are generally meant to group/wrap other HTML Elements inside of them.
    • These tags don't display anything by themselves.
    • Eg : div tag, span tag, footer tag
